## Markapp: A list of HTML libraries

[Original](https://lea.verou.me/blog/tags/original "81 posts with this tag"), [JS](https://lea.verou.me/blog/tags/js "aka Javascript, 55 posts with this tag"), [HTML](https://lea.verou.me/blog/tags/html/ "12 posts with this tag"), HTML Libraries

[![Screen Shot 2016-08-26 at 17.09.24](https://lea.verou.me/2016/08/markapp-a-list-of-html-libraries/images/Screen-Shot-2016-08-26-at-17.09.24.png)](https://lea.verou.me/2016/08/markapp-a-list-of-html-libraries/images/Screen-Shot-2016-08-26-at-17.09.24.png)I have often lamented how many JavaScript developers don’t realize that a large percentage of HTML & CSS authors are not comfortable writing JS, and struggle to use their libraries.

To encourage libraries with HTML APIs, i.e. libraries that can be used without writing a line of JS, I made a website to list and promote them: [MarkApp](https://projects.verou.me/markapp). The list is currently quite short, so I’m counting on you to [expand it](https://github.com/LeaVerou/markapp). Seen any libraries with good HTML APIs? Add them!
